1. strong
    • Use cases: To emphasize important text and make it bold.
    • Best practices: Use sparingly and only to highlight truly important content. Avoid using for purely stylistic purposes.
  2. em
    • Use cases: To indicate emphasized text and make it italic.
    • Best practices: Use sparingly and only to indicate genuine emphasis. Avoid using for purely stylistic purposes.
  3. u
    • Use cases: To underline text, typically used to indicate hyperlinks.
    • Best practices: Use only for hyperlinks, and avoid underlining text for emphasis.
  4. sup and sub
    • Use cases: To create superscript and subscript text, respectively.
    • Best practices: Use only for mathematical or scientific notation, such as exponents or chemical formulas.
  5. blockquote
    • Use cases: To indicate a block of quoted text.
    • Best practices: Use for quotes, such as from a book or article. Use CSS styling for purely decorative purposes.
  6. small
    • Use cases: To indicate small text, typically used for disclaimers or footnotes.
    • Best practices: Use for small text only, not for purely decorative purposes.
