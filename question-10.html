  • simple and easy to learn

  1. CSS

Primary Use Case Key Features/Benefits
to give design or style the website
  • Consistency: CSS allows developers to create a consistent look and feel across multiple pages of a website. This is because styles can be defined in a single file and applied to multiple HTML pages. This saves time and effort and ensures that the website has a consistent design.
  • Flexibility: CSS provides a high degree of flexibility when it comes to designing web pages. It allows developers to control the layout, colors, fonts, and other visual elements of a web page. This flexibility makes it easier to create responsive designs that adapt to different screen sizes and devices.

  1. JS

Primary Use Case Key Features/Benefits
to give a dynamic web page instruction
  • Client-side validation: JavaScript can be used to validate user input on the client-side, which can reduce the number of requests sent to the server and improve the performance of the web application. It can also help prevent incorrect or malicious data from being submitted to the server.
  • easy to use
  • Improved user experience: By adding dynamic elements and animations to a web page, JavaScript can help improve the user experience and make the web application more visually appealing.
