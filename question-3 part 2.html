HTML Elements:

An HTML element is a piece of content on a web page that is defined by a start tag, content, and an end tag. The start tag begins an HTML element and the end tag closes it. Elements can contain other elements, attributes, and text.

<p> this is paragraph element </p>

HTML Attributes:

An HTML attribute is a modifier of an HTML element. Attributes provide additional information about an element, such as its behavior, appearance, or name. Attributes are included in the opening tag of an element and consist of a name and a value.

<a href = "link"> </a>

HTML TAGS

. HTML tags are enclosed in angle brackets, and usually come in pairs consisting of a start tag and an end tag. The start tag indicates the beginning of an HTML element, and the end tag indicates the end of that element. For example, the < p > tag is used to define a paragraph of text, and consists of a start tag < p > and an end tag </p > . The content between the start and end tags is the text that appears on the web page as a paragraph. HTML tags can also include attributes, which provide additional information about the element. Attributes are included within the start tag and consist of a name and a value. For example, the tag is used to create links, and includes an href attribute to specify the URL that the link points to.
